# 🏓 Classic Pong Game - Python implementation
A recreation of the legendary **Pong Game** developed as part of my computer science fundamentals and software engineering journey. This project demonstrates real-time GUI handling and basic physics logic.

## 🚀 Technical Highlights
This project isn't just a game; it's an application of several engineering and programming concepts:
* **Real-time Graphics:** Utilizing the `turtle` library for rendering shapes and movements.
* **Collision Detection:** Implementing coordinate-based logic to handle ball-to-paddle and ball-to-wall collisions.
* **Event-Driven Programming:** Mapping keyboard inputs (`W`, `S`, `Up`, `Down`) to trigger specific function calls.
* **Frame Rate Control:** Using `.tracer(0)` and `.update()` to manage manual screen refreshing for smoother performance.

## 🛠️ Features
* **Two-Player Mode:** Split controls for local multiplayer.
* **Dynamic Scoreboard:** Real-time scoring updates at the top of the screen.
* **Boundary Physics:** Automatic ball reflection on top/bottom walls and score resets on horizontal out-of-bounds.

## 🕹️ How to Play
1. **Player 1 (Left - Blue):** Use `W` to move up and `S` to move down.
2. **Player 2 (Right - Red):** Use `Up Arrow` to move up and `Down Arrow` to move down.
3. First to reach a high score wins!
